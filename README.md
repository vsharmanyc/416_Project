**Description:** 

Full Stack web application displaying congressional voting data for different voting districts.

**Run Project:**
* Install Node.js and NPM
* ```git clone https://github.com/vsharmanyc/416_Project.git```
* ```cd 416_Project```
* ```npm install```
* ```npm start```
